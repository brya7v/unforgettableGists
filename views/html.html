HTML

HTML
  • Hyper Text Markup Language.
  • Described by HTML tags wich describe different types of content, headers, paragraphs, links, etc.
HTML elements
  • An HTML element is defined by a start tag, some content, and an end tag.
  • <tagname> Content goes here... </tagname>
  • Elements can have attributes, like classes, src, alt, etc
script
  • Scripts are appendend at the end of the body, before the closing tag (order matters), its commonly used props are: src
  • It's are NOT selfclosing element <script src=""> </script>.
link
  • The link element allows authors to link their document to other resources.
  • It's ARE a selfclosing element <link hfref="" rel:"" >.
  • It must contain a href attribute wiwhich must be present and must contain a valid non-empty URL potentially surrounded by spaces.
  • It must contain a rel attribute, which, if present, must have a value that is a set of space-separated tokens. 'stylesheet'
picture - srcset
  • Dynamic images depending on media and srcset, also size can be increased or decreased by an X modifier
  • It must have the <source srcset:"" media:""> and <img alt=""> elements to work properly. 
        <picture>
            <source srcset="smoll.png 0.8x" media="(min-width:1000px)"> ( / 0.8 smaller (20% size inc))
            <source srcset="big.png 2x" media="(max-width:1000px)"> ( / 2x smaller (half size))
            <img>
        </picture>
Basic HTML template 
    <!DOCTYPE html>
    <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="css/style.css">
            <title>Document</title>
        </head>
        <body>
            <main>
                <nav>
                    <header>Unforgettable Gists</header>
                    <div class="nav">
                        <div class="nav_item nav_item-item3" section="item3">item3</div>
                        <div class="nav_item nav_item-item4" section="item4">item4</div>
                        <div class="nav_item nav_item-item5" section="item5">item5</div>
                    </div>
                </nav>
                <section class="section section-item3">
                    <div>ITEM 3</div>
                </section>
                <section class="section section-item4">
                    <div>ITEM 4</div>
                </section>
                <section class="section section-item5">
                    <div>ITEM 5</div>
                </section>
            </main>
            <script src="js/main.js"></script>
        </body>
    </html>
